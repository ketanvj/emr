First Create Event Bus
java -jar selenium-server-4.1.1.jar  event-bus

Session Map
java -jar selenium-server-4.1.1.jar sessions

New Session Queue
java -jar selenium-server-4.1.1.jar sessionqueue

Distributor
java -jar selenium-server-4.1.1.jar distributor --sessions http://localhost:5556 --sessionqueue http://localhost:5559 --bind-bus false

Router
java -jar selenium-server-4.1.1.jar router --sessions http://localhost:5556 --distributor http://localhost:5553 --sessionqueue http://localhost:5559

Create Nodes
java -Dwebdriver.chrome.driver="" -Dwebdriver.gecko.driver="" -jar selenium-server-4.1.1.jar node
